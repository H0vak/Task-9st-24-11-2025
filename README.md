Mathematical Function Calculations (C++)
Цей проєкт реалізує програму для обчислення складних математичних функцій за заданими параметрами $a$, $b$ та $x$. Робота демонструє навички декомпозиції коду на окремі функції та використання тригонометричних і степеневих операцій.

Технології:
C++.

Бібліотека <cmath> — використана для доступу до математичних функцій, таких як sin(), cos(), pow() (піднесення до степеня) та sqrt() (корінь квадратний).

Бібліотека <windows.h> — для забезпечення коректної підтримки кириличного тексту в консольному інтерфейсі.

Що робить ця програма:

Декомпозиція коду: Обчислення значень y та z винесено в окремі функції calcY та calcZ, що покращує читабельність та структуру проєкту.

Обробка формул:
  Функція y: Реалізує обчислення куба синуса від квадрата суми x^2 + a та віднімання квадратного кореня відношення x/b.
  Функція z: Реалізує обчислення суми відношення x^2/a та куба косинуса суми x + b.

Робота з типом double: Забезпечує високу точність обчислень для дробових значень аргументів та результатів.
